必须要安装 node 以及 全局 安装 gulp npm install gulp 然后再在项目里面 安装 gulp npm install gulp --save 然后再在项目里面 安装 项目所需依赖 npm install 这里不建议用 （cnpm install） 因为经测试 会有很多相关依赖安装不成功的情况而引起的报错。 开发环境： 建议用 这个命令：npm run dev 原因 跟 vue的脚手架 开发环境的命令是 一模一样的 打包项目： 建议用 这个命令：npm run build （npm run build 会压缩你的代码的） 原因 跟 vue的脚手架 生产环境的命令是 一模一样的
