利用 Chrome 浏览器 + 真机进行移动端 debug

阅读 1281
收藏 95
2017-09-11
原文链接：hxlovexc.com
移随着移动端的不断推进, 特别是安卓 机型, 分辨率, 系统版本存在很多分类, 今天教大家如何利用Chrome进行手真机调试 。
1、先打开谷歌(或者谷歌内核)浏览器, 调出控制台
2、然后点击右侧工具栏


依次选择 More tools -> Remote devices 然后会出现一个调试窗口,


3、然后手机数据线连接电脑 ( 注意:  第一次连接的时候手机会弹出权限请求, 一定要点击确定哦);

如果连接成功调试窗口上就会显示出你连接的手机 - 我的是魅族 PRO 6 PLUS， 图片上已经显示了





4、连接成功后点击一下你的手机型号, 然后用手机打开你需要调试的网页 - 打开后控制台上会显示你打开的页面地址, 然后点击右侧的 inspect

这个时候会弹出一个熟悉的窗口, 这个就是Chrome的控制台啦, 然后就可以和pc一样的自由的debug咯~~




ps: 我绝对不会告诉你如果没有显示出页面可以点击右侧第一个按钮!


今天就写到这啦, 这个方案适合安卓平台, 苹果的Safari是不适合的哦!
