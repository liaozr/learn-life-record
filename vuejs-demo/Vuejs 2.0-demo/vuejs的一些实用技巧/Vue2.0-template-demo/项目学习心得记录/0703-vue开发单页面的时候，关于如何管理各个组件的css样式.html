拿 vue来做单页面开发的时候，由于是打包后各个组件都会被放到一起的， 所以，如果各个组件间命名规范不好的话，很容易引起样式的冲突。 所以，这对于良好的管理css样式 就凸显的异常重要了。 这里，给的建议是，就拿sass来开发css样式 各个组件中，各个.vue文件中 的style标签里面的样式 都得写成局部的，也就是说都只能在这个组件里面使用的 只需在 style标签 加上 scoped即可，如下所示即可，即表明里面的样式那些，只能在这个组件里面使用。 觉得组件里面的这个模块 的样式可以复用的话，就不建议写到 style标签里面了， 可以将它提取出来，放到scss文件中，供全局使用，所以，scss里面的css样式 应该都是全局的，都是些比较能复用的一些css样式。 各个组件里面的样式，不能复用的样式那些就写到各个组件中，用scoped属性 括起来 表明只能在 这个组件里使用这些样式。
