----------------------------------------------------------------------------------------------------- 首先我在data函数里面进行定义定时器名称： data() { return { timer: null // 定时器名称 } }, 然后这样使用定时器： this.timer = (() => { // 某些操作 }, 1000) 最后在beforeDestroy()生命周期内清除定时器： beforeDestroy() { clearInterval(this.timer); this.timer = null; } -----------------------------------------------------------------------------------------------------
